# ALCUK-PracticeChecker
알쿡 과제 채점프로그램
www.acmicpc.net 사이트에서 연습 현황을 긁어 붙여넣으면 입력한 커트라인에 따라 과제 수행여부를 판단하고 반별로 나누어 미달자를 출력해줌
